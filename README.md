Here are some details about my setup:

- Shell : fish
- Terminal : alacritty
- Editor : neovim/neovide
- Panel : polybar
- Compositor : picom
- Notify Daemon : dunst
- Application Launcher : rofi
- File Manager : ranger/yazi
- Wallpaper Manager : feh
